# 🚀 CA Final MCQ Assessment Tool

A powerful web-based MCQ testing platform designed for CA Final students.  
Practice AI-generated questions, attempt timed tests, track performance, and analyze detailed reports — now with optional **user login support (Firebase)**.

---

## 🌐 Live Demo

👉 https://hardikselva6-rgb.github.io/CA-Final-MCQ-Tool-Kit/

---

## ✨ Features

### 📋 Core Features
- ✅ Paste AI-generated MCQs (tab or pipe format)
- ✅ Timed test environment
- ✅ Smart question navigation panel
- ✅ Skip & revisit questions
- ✅ Auto evaluation upon submission

---

### 📊 Performance Analytics
- ✅ Score percentage
- ✅ Accuracy tracking
- ✅ Time spent per question
- ✅ Weak area identification
- ✅ Question-wise performance
- ✅ Speed vs accuracy analysis
- ✅ Average time analysis

---

### 🔍 Review System
- ✅ Correct vs wrong answers highlighted
- ✅ Detailed per-question analysis
- ✅ Time spent on each question

---

### 👤 (NEW) User System – Firebase Ready
- ✅ Email signup & login support
- ✅ Save test results (future upgrade)
- ✅ Track performance over time
- ✅ Works without backend server

---

## 📂 How to Use

1. Open the tool in browser
2. Paste MCQ questions in supported format
3. Enter test name and your name
4. Choose time limit
5. Start the test
6. Submit and view detailed analytics
